以下操作均在`org-agenda`中的`agenda for current week or day`视图下快速编辑：


| 操作 | 快捷键 |
| ---- | ---- |
| 变更任务状态 | t |
| 添加备注 | z |
| 添加日记 | i |
| 编辑note | z |
| 重建agenda | r |
| 打开日历 | a |
| 下周任务列表 | f |
| 上周任务列表 | b |
| 打开任务所在原始文件 | enter |
