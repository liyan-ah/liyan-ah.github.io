AliceBot

Lucia-lang

一个受到了 Lua 和 Python 语言启发的轻量级、可嵌入的动态强类型编程语言。使用 Rust 语言实现了完整的词法分析、语法分析、语义分析、字节码生成和支持 GC 的基于栈的虚拟机。具有错误处理、头等函数、匿名函数、闭包、迭代器、运算符重载、安全导航运算符、尾调用优化等现代语言特性。
